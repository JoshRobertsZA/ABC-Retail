## üßæ How to Download and Open the POE

### Step 1: Download the Project from GitHub

1. [Click here to view the repository](https://github.com/JoshRobertsZA/CLDV6212POE)

2. Click the big green **Code** button.

3. Click **Download ZIP**.

4. When it finishes downloading, go to your **Downloads** folder.

5. **Right-click** the ZIP file and choose **Extract All...**  
   (Follow the steps to unzip it into a normal folder.)

---

### Step 2: Open the Project in Visual Studio 2022

1. Open **Visual Studio 2022** (you can find it in your Start Menu).

2. On the home screen, click **"Open a project or solution"**.

3. Go to the folder you just unzipped.

4. Find and click on the file that ends with **`.sln`** (example: `CLDV6212POE.sln`), then click **Open**.

---


### If You Get an Error
